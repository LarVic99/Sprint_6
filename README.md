🛴 Автотесты для сервиса «Яндекс.Самокат» (Sprint 6) Этот проект — автотесты UI-интерфейса учебного веб-приложения «Яндекс.Самокат» (https://qa-scooter.praktikum-services.ru/), написанные с использованием Selenium, pytest и Allure. В проекте используется структура с Page Object Model для удобного и масштабируемого тестирования.

Структура проекта выглядит следующим образом: есть папки locators с локаторами элементов для разных страниц, папка pages с описанием страниц и их методов, папка tests, в которой лежат тесты, разделённые по функциональности, файл conftest.py с фикстурой запуска браузера, requirements.txt для зависимостей и README.md с описанием проекта. Тесты покрывают основные сценарии: проверку вопросов-ответов в разделе FAQ, позитивный сценарий заказа самоката с двумя наборами данных и двумя точками входа, проверку переходов по логотипам Самоката и Яндекса, при этом используется параметризация для удобства и расширяемости тестов.

Чтобы запустить проект, сначала нужно установить зависимости командой pip install -r requirements.txt, затем запустить тесты командой pytest --alluredir=allure_results и после завершения тестов сгенерировать и открыть Allure-отчёт командой allure serve allure_results. В проекте используется Firefox в headless режиме, что позволяет запускать браузер без открытия графического интерфейса, а фикстура browser в conftest.py автоматически поднимает и закрывает браузер для каждого теста.

Данный проект был создан в рамках учебного курса Яндекс.Практикум, спринт 6, и позволяет закрепить навыки UI-тестирования с помощью Selenium, параметризацию тестов, использование Page Object Model и генерацию красивых отчётов в Allure.
